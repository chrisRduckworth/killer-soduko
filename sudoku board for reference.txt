box position is defined by x and y coords

if 0 <= x % 9 < 3, x is at start
   3 <= x % 9 < 6, x is at middle
   6 <= x % 9 < 9, x is at end
so x adjustable is Math.floor((n%9)/3)

if 0 <= y < 27 y is 0
  27 <= y < 54 y is 3
  54 <= y < 81 y is 6 

so y adjustable is 3 * Math.floor(n / 27)
